:: MaxLink 0.36 installation instructions for Mac OS X ::


:: For Processing ::

   Drag the "maxlink" folder into your Processing libraries folder. (You may need to create this folder.)
      (~/Documents/Processing/libraries).


:: For Max/MSP 4.x ::

   Drag the "for cycling '74 java lib/maxlink.jar" jar file into the /Library/Application Support/Cycling '74/java/lib folder.


:: For Max/MSP 5.x ::

   Drag the "for cycling '74 java lib/maxlink.jar" jar file into the /Applications/Max5/Cycling '74/java/lib folder.
		

Restart both apps.  

If you have a firewall running, enable network traffic on port 7474.

More info: http://jklabs.net/maxlink/
